## Persona A: 직장인 김현우

**"믿을 수 있는 정보로 퇴근 후 2시간을 써서 월급 외 수입을 만들고 싶다"**

| 항목 | 내용 |
|------|------|
| 나이 | 32세 |
| 직업 | 중소기업 마케터 |
| 거주지 | 서울/수도권 |
| 월 소득 | 280만원 (세후) |
| 목표 추가 수입 | 월 30~50만원 |
| 가용 시간 | 평일 퇴근 후 2~3시간, 주말 오전 |

### Pain Points
- 검색할 때마다 유료 강의 결제 유도 페이지로 연결됨
- "진짜 얼마 버는지" 검증된 정보를 찾기 어려움
- 초기 투자 비용 있는 부업에 대한 두려움 (실패 리스크)
- 어디서부터 시작해야 할지 모르는 막막함

### 행동 패턴
- 정보 탐색 채널: 구글 검색 > 유튜브 > 블로그
- 신뢰 기준: 광고가 없는가, 실제 사례가 있는가
- 의사결정 속도: 느림 (비교 후 신중하게 선택)

### 우리 플랫폼에서 원하는 것
1. 난이도·초기 비용이 명확히 표시된 부업 정보
2. "퇴근 후 2시간으로 가능한가?" 현실적인 판단 근거
3. 광고·강의 유도 없는 순수 정보

## Persona B: 대학생 이서연

**"AI 툴로 할 수 있는 새로운 부업이 뭔지 알고 싶다. 돈 없이 시작할 수 있는 것으로."**

| 항목 | 내용 |
|------|------|
| 나이 | 22세 |
| 상태 | 취준생 + 편의점 알바 병행 |
| 거주지 | 지방 대학가 (기숙사) |
| 월 소득 | 알바비 70만원 |
| 목표 | 알바 대체 또는 보완 가능한 온라인 수입원 |
| 가용 시간 | 방학 중 자유롭게, 학기 중 주 10~15시간 |

### Pain Points
- "무자본 부업"으로 검색하면 피싱·MLM 링크만 나옴
- AI 툴에 관심은 있지만 어떤 부업에 어떻게 적용하는지 연결점 모름
- 성공 사례들이 "재능 있는 사람만 되는 것" 같아 보임
- 시작 가이드가 너무 추상적이거나 반대로 너무 복잡함

### 행동 패턴
- 정보 탐색 채널: 틱톡 > 인스타 릴스 > 유튜브 쇼츠 > 구글 검색
- 신뢰 기준: 또래 성공 사례, 커뮤니티 반응
- 의사결정 속도: 빠름 (일단 해보고 아니면 그만)

### 우리 플랫폼에서 원하는 것
1. "AI 활용", "무자본" 태그로 빠르게 필터링
2. 당장 오늘 시작할 수 있는 첫 단계 가이드
3. 예상 수익의 현실적인 범위 (과장 없이)

## 공통 인사이트

- **두 페르소나 모두**: 사기·과장 광고에 지쳐 있음 → **신뢰감이 첫 번째 허들**
- **두 페르소나 모두**: 정보를 받고 나서 "나도 할 수 있겠다"는 자신감이 핵심 전환 포인트
- **차이점**: 김현우는 안정성·검증 중시, 이서연은 새로움·속도 중시
